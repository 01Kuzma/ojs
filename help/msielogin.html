Login Problems with Microsoft Internet Browser

Several users have experienced problems logging in using Internet Explorer

Solutions

  1. The quick solution:
    Avoid using Internet Explorer. Instead try using a different browser such as Firefox, Chrome, Safari or Opera
  2. Fix the login-issue with Internet Explorer:
    You can try to add the journal URL (web-page-address) to Trusted sites in Internet Explorer (microsoft support page).
    Instructions:
    1. Open Internet Explorer by clicking the Start button, and then clicking Internet Explorer.
    2. Navigate to the website that you want to add to a specific security zone.
    3. Click the Tools button or, gears icon and then click Internet Options
    4. Click the Security tab, and then click a security zone (Local intranet, Trusted sites, or Restricted sites).
    5. Click Sites
    6. If you clicked Local intranet in the previous step, click Advanced.
    7. The website should be shown in the Add this website to the zone field. Click Add.
    8. If the site is not a secure site (HTTPS), clear the Require server verification (https:) for all sites in this zone check box.
    9. Click Close, and then click OK (or click OK twice if you clicked Local intranet in step 4).

Innloggingsproblemer med Microsoft Internet Explorer

Flere brukere har hatt problemer med å logge inn til tidsskriftet med Internet Explorer

Løsninger

  1. Den raske løsningen:
    Unngå Internet Explorer. Prøv i stedet en annen nettleser som Firefox, Chrome, Safari eller Opera
  2. Fiks innloggingsproblemet med Internet Explorer:
    Prøv å legge til tidsskriftets nettstedsadresse som sikkert område i Internet Explorer. (microsoft supportside, engelsk)
    Fremgangsmåte:
    1. Åpne Internet Explorer.
    2. Velg tannhjulet øverst i høyre hjørne
    3. Velg alternativer for Internett
    4. Velg fanen Sikkerhet
    5. Trykk på Klarerte områder
    6. Velg knappen Områder og legg til URL (webadressen) til tidsskriftet
